## Что делает:
- Программа выводит графические стрелы(<-) в указанном кол-ве, состоящих из символов '+'.
## Как запустить:
- Двойной щелчок левой кнопки мыши по файлу start.bat
## Содержимое папки:
- Arrows.class - сама программа на языке Java
- Arrows.java - исходный код программы
- start.bat - удобный способ запуска для пользователя
